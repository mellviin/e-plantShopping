# e-plantShopping

e-plantShopping is a web-based plant shopping application that allows users to browse plants, add them to a shopping cart, and manage their selections easily. The application provides a simple and user-friendly interface for online plant purchasing.

## Project Overview

The e-plantShopping application is designed to simulate an online plant store where users can:
- View a list of available plants
- Add plants to the shopping cart
- Increase or decrease item quantities
- View the total cost of selected items

This project is built using modern front-end technologies and focuses on clean UI design, state management, and component-based architecture.

## Technologies Used
- React
- Redux Toolkit
- JavaScript
- HTML & CSS

## Purpose
This project helps demonstrate front-end development skills using React and Redux, including state management, component structure, and user interaction handling.
